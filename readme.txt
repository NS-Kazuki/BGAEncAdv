　BGAEncAdvance　Ver0.035

　bmsファイルを解析し、BGAを動画形式で書き出します。
　以下の特徴があります。(若干信用ならないところもあります)
・D&Dで一発変換
・基本的なbmp・png・jpgのサポート
・本来ならはみ出すBGA領域も描画可能
・#BMPと#BGAを解析しての動画サイズ自動取得
・LAYER2のサポート
・#ARGBのサポート
・#BGAのサポート
・動画ファイルのサポート(何気に動画も#ARGBと#BGAに対応しています)
・音声ファイルの合成(音声データの生成は行わないため、bmx2wav等でご用意ください)

　以下、予定
・レイヤー合成の高速化(やれるだけのことはやったつもり)

　以下、現時点での注意点
・出力可能なコーデックはHuffyuv(可逆圧縮)のみ

　以下、どうしようもないもの
・はみ出しBGAと"#BGA"を両方使ったBMSに対応させると、256より大きいサイズでBGAを作る場合に"#BGA"の座標を直感的に扱えなくなる
・D&Dからの標準入力で、"ー"や"ヽ"等の文字を使うとバグる(環境依存?)

　以下、特に対応させる予定がないもの
指数表記の解釈(出来なくもないけど意図的に記述する人がいなさそうなので誰得)
